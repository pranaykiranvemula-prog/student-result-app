# student-result-app
A Flask-based web application that allows users to enter student marks and automatically calculates total, percentage, and grade. The app displays student results in a simple and organized format.
# Student Result Management System - Python Flask Web App
A full-stack web application built with Python Flask for managing
student marks and generating grade reports.

## Features
- Add students with marks for 5 subjects
- Auto-calculates total, percentage and grade for each student
- Dashboard with class summary (total, passed, failed, average)
- Colour-coded grade badges (A+, A, B, C, D, FAIL)
- Delete students from the system
- Download full results as CSV file
- Clean, responsive web interface
## Tech Stack
- Backend: Python 3, Flask
- Frontend: HTML5, CSS3
- Storage: JSON file
## How to Run
1. Install Flask: pip install flask
2. Run the app: python app.py
3. Open browser: http://localhost:5000
## Grade Scale
A+ = 90%+ | A = 80-89% | B = 70-79%
C = 60-69% | D = 50-59% | FAIL = below 50%
